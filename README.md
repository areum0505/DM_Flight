# DM Flight
비행기를 조종해 적을 격파하는 클래식 아케이드 게임입니다.
초반에는 약한 무기를 장착하여 액션쾌감을 느끼기 어렵지만, 시간이 갈 수록 점차 강화되는 무기와 생동감 넘치는 보스전을 통해 도파민과 함께 액션 쾌감을 느낄 수 있습니다.

# 구조
DM_Flight/
    ├───index.html              # 메인 HTML 파일
    ├───README.md               # 프로젝트에 대한 설명을 기록하는 문서
    ├───sketch.js               # p5.js의 메인 스케치 파일로, 게임의 초기 설정과 메인 루프를 담당
    ├───style.css               # 스타일을 정의하는 CSS 파일
    ├───components/             # 게임에 등장하는 핵심 요소(객체)들의 클래스 정의
    │   ├───Boss.js             # 보스 캐릭터의 동작, 속성 등을 정의
    │   ├───Bullet.js           # 총알의 동작, 속성 등을 정의
    │   ├───Enemy.js            # 일반 적 캐릭터의 동작, 속성 등을 정의
    │   ├───Item.js             # 게임 아이템 (젬, 코인 등) 의 동작, 속성 등을 정의
    │   └───Player.js           # 플레이어 캐릭터의 동작, 속성 등을 정의
    ├───configs/                # 게임의 각종 설정을 관리하는 파일
    │   ├───config.js           # 게임의 기본 설정 (예: 캔버스 크기, 프레임률 등)
    │   ├───enemyConfig.js      # 적 캐릭터의 종류, 체력, 공격 패턴 등 상세 설정
    │   └───waves.js            # 적들이 등장하는 순서와 패턴(웨이브)을 정의
    ├───managers/               # 게임의 특정 기능들을 전문적으로 관리하는 매니저 클래스
    │   ├───SceneManager.js     # 게임의 여러 장면(시작, 게임 중, 게임 오버)을 전환하고 관리
    │   └───SpawnManager.js     # 적이나 아이템의 생성을 관리
    ├───scenes/                 # 게임의 각 장면(Scene)에 대한 클래스 정의
    │   ├───GameOverScene.js    # 게임 오버 화면의 UI와 로직을 담당
    │   ├───GameScene.js        # 실제 게임 플레이가 이루어지는 화면의 로직을 담당
    │   └───StartScene.js       # 게임 시작 화면의 UI와 로직을 담당
    └───ui/                     # UI 관련 요소들을 보관할 수 있는 곳
